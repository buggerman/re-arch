Opinionated Arch Linux Desktop Installer

Boot from Arch Linux ISO, connect to internet, and run one command.
Get production-ready Arch Linux with snapshots, optimizations, and four desktop environments.

🚀 Quick Install

⚠️ Prerequisites: Boot from Arch Linux ISO and connect to internet first

curl -fsSL https://re-arch.xyz/install | bash

That's it! Choose your desktop environment and let it install.
For local testing: ./test-install or bash install

What You Get

A carefully curated system optimized for developers and power users

🛡️

Atomic Snapshots

Automatic Btrfs snapshots with GRUB integration. Rollback your entire system in 30 seconds if something breaks.

🚀

Performance First

Linux Zen kernel, zram compression, process optimization, and low-latency PipeWire audio.

🔒

Security by Default

Firewall enabled, Flatpak sandboxing for applications, and minimal attack surface out of the box.

📦

Smart Package Management

Four-tier system: pacman for core, Flatpak for apps, AUR when needed, Homebrew for dev tools.

⚠️ Important Warning

This will completely erase the target disk. Always test in a virtual machine first!

📋 System Requirements

  • ● Boot from Arch Linux ISO (latest stable)
  • ● 4GB+ RAM (8GB+ recommended)
  • ● 20GB+ storage space
  • ● UEFI boot mode
  • ● Stable internet connection
  • ● Fresh installation (existing data will be erased)

🔧 Manual Installation (If Single Command Fails)

⚠️ Prerequisites: Boot from Arch Linux ISO and connect to internet first

If the one-command installer doesn't work, you can install manually:

Step 1: Run archinstall

# Choose your desktop environment (kde, gnome, xfce, or hyprland) archinstall --config-url https://re-arch.xyz/config-kde.json --creds-url https://re-arch.xyz/creds.json

Step 2: Run post-installation script

# After archinstall completes, run in chroot: arch-chroot /mnt curl -fsSL https://re-arch.xyz/re-arch-lite.sh | bash exit # Then reboot: umount -R /mnt reboot

Desktop Options: Replace config-kde.json with config-gnome.json, config-xfce.json, or config-hyprland.json

🏃‍♂️ After Installation

# Default login: user/rearch (change password immediately) passwd # Install web browser flatpak install flathub org.mozilla.firefox # Update system sudo pacman -Syu # Install development tools (after shell restart) brew install git curl nodejs
