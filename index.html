The Re-Arch Procedure

Transform minimal Arch Linux into an optimized desktop powerhouse

One Command Installation

curl -fsSL https://raw.githubusercontent.com/buggerman/re-arch/main/re-arch.sh | bash

Run this command as root in your chrooted Arch Linux environment

⚠️ Critical Warning

This script makes extensive system-wide changes. Only run on fresh, minimal Arch Linux installations with Btrfs root filesystem.

Complete Installation Workflow

From bare metal to production desktop in minutes

1

Install Arch Linux

Use archinstall with our provided configuration for automated base system setup with Btrfs.

2

Chroot Environment

Boot from installation media, mount your system, and chroot into the installed environment.

3

Run Re-Arch

Execute our one-command installer. The script handles everything else automatically.

4

Reboot & Enjoy

Reboot into your optimized system with KDE Plasma, snapshots, and performance tuning.
