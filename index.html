üîß Manual Installation (If Single Command Fails)

‚ö†Ô∏è Prerequisites: Boot from Arch Linux ISO and connect to internet first

If the one-command installer doesn't work, you can install manually:

Step 1: Run archinstall

# Choose your desktop environment (kde, gnome, xfce, or hyprland) archinstall --config-url https://re-arch.xyz/config-kde.json --creds-url https://re-arch.xyz/creds.json

Step 2: Run post-installation script

# After archinstall completes, run in chroot: arch-chroot /mnt curl -fsSL https://re-arch.xyz/re-arch-lite.sh | bash exit # Then reboot: umount -R /mnt reboot

Desktop Options: Replace config-kde.json with config-gnome.json, config-xfce.json, or config-hyprland.json
