Opinionated Arch Linux Desktop Installer

Professional one-command installer for production-ready Arch Linux.
Get snapshots, optimizations, and your choice of desktop environment.

⚠️
Prerequisites Required
Boot from Arch Linux ISO and connect to internet first

What You Get

A carefully curated system optimized for developers and power users

🛡️

Atomic Snapshots

Automatic Btrfs snapshots with GRUB integration. Rollback your entire system in 30 seconds if something breaks.

🚀

Performance First

Linux Zen kernel, zram compression, process optimization, and low-latency PipeWire audio.

🔒

Security by Default

Firewall enabled, Flatpak sandboxing for applications, and minimal attack surface out of the box.

📦

Smart Package Management

Four-tier system: pacman for core, Flatpak for apps, AUR when needed, Homebrew for dev tools.

Installation

Multiple ways to install, from one-command to manual control

🚀 Quick Install

One command to production-ready system

curl -fsSL https://re-arch.xyz/install | bash

That's it! Choose your desktop environment and let it install.

🔑
Default Login Credentials
Username: user | Password: rearch
Change these immediately after first login for security!

1 Run archinstall

Choose your desktop environment (kde, gnome, xfce, or hyprland):

archinstall --config-url https://re-arch.xyz/config-kde.json --creds-url https://re-arch.xyz/creds.json

2 Post-installation setup

After archinstall completes, run the post-config script:

arch-chroot /mnt curl -fsSL https://re-arch.xyz/re-arch-lite.sh | bash exit && umount -R /mnt && reboot
📝
Desktop Options
Replace config-kde.json with config-gnome.json, config-xfce.json, or config-hyprland.json
